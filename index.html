ENKRIPTOR

MENDUKUNG SEMUA JENIS FILE

PANDUAN PENGGUNA

1. Master Password / Passphrase

Master password adalah kunci utama untuk membuka atau mengenkripsi file kamu.

Penting: Jika lupa master password, file terenkripsi tidak bisa dibuka lagi.

Gunakan passphrase panjang, mudah diingat, tapi sulit ditebak, contoh:

kopi-pagi-hujan-motor-kenangan-2019

2. Membuka atau Membuat File Enkripsi

A. Membuka File Enkripsi Lama

  1. Masukkan master password di kolom “Master Password”.
  2. Klik Pilih File .secret untuk memilih file terenkripsi yang sudah ada.
  3. Klik Buka / Buat File Enkripsi.
  4. Jika password benar, file asli akan ditampilkan di panel preview.

B. Membuat File Enkripsi Baru

  1. Masukkan master password di kolom yang tersedia.
  2. Jika tidak memilih file, klik Buka / Buat File Enkripsi.
  3. File baru akan dibuat otomatis dengan master password yang dimasukkan.

3. Menggunakan QR Code untuk Master Password

A. Dari Gambar QR

  1. Klik Pilih Gambar pada bagian di samping tombol scan via kamera.
  2. Pilih file gambar QR.
  3. Master password akan otomatis terisi dari QR code.

B. Dari Kamera

  1. Klik Scan QR via Kamera.
  2. Arahkan kamera ke QR code.
  3. Jika berhasil, master password muncul otomatis.
  4. Kamera akan mati otomatis setelah QR terbaca, atau klik tombol untuk mematikan kamera.

4. Menambahkan dan Mengenkripsi File

  1. Pilih file untuk dienkripsi dengan klik Tambahkan File.
  2. File akan ditampilkan di panel preview dengan nama, tipe, dan ukuran.
  3. Klik Simpan & Enkripsi untuk mengenkripsi file menggunakan master password.
  4. File terenkripsi otomatis didownload dengan ekstensi .secret.

5. Mengunduh File Hasil Dekripsi

  1. Jika sudah membuka file terenkripsi atau menambahkan file, klik Unduh File di panel file viewer.
  2. File akan didownload dalam format aslinya.
  3. Pastikan master password benar, jika tidak file tidak bisa didekripsi.

6. Keamanan & Auto-Lock

  • Semua proses enkripsi/dekripsi terjadi di perangkat, data tidak dikirim ke server manapun.
  • File akan otomatis terkunci setelah 2 menit tidak ada aktivitas (mouse atau keyboard).
  • Setelah terkunci, masukkan master password lagi untuk membuka file.

7. Tips Tambahan

  • Gunakan passphrase panjang dan unik.
  • Simpan beberapa salinan .secret di lokasi berbeda.
  • Backup master password, misalnya via QR code.
  • File hanya berisi file terenkripsi, bukan teks langsung.
